# (Placeholder for Project Name)

## (Placeholder) is a link management hub, built to streamline the connection between individual users, companies and content creators on social media platforms to their respective links of interest. Embedded within a their profile, users have the freedom to bundle all their links into one accessable link, while making an environmental impact compared to compeditors like LinkTree

## Features

- URL Compression (using 'crypto' library)
- User Accounts stored within SupaBase
- Publically available Dashboards
- Email-Verification Signup

## Technologies Used (FOR DEMO)

- Backend: Node.js: https://nodejs.org/, Express.js: https://expressjs.com/, REST API + Routing
- Frontend: HTML, JS
- Session Management: 'express-session' for persistance
- Environment Handlers: dotenv 
- Hosting: TBD (Render?)

## Future Enhancements

- Real-Time Analytics with WebScraping Protection
- Social-system for dashboards (Descriptions of projects and the ability to tag other users)
- Sustainability integrations for high-traffic accounts to support environmental causes
